Creando un repositorio en Github

  1. Crear el repositorio en Github<. www.github.com
  2. Crear carpeta donde se almacenar√° nuestro proyecto
  3. Abrir la CARPETA(no un archivo) con visual studio coda
  4. Abrir la terminal integrada
  5. git initeste comando inicializa el repositorio
  6. git remote add origin https://github.com/nombredeusuario/repositorio.git esta instruccion agrega el origen
  7. git add nombredearchivo nuestros cambios a stage, o nuestra piscina de cambios
  8. git commit -m "nombre del commit" agrega un nombre a nuestra piscina de cambios
  9. git push -u origin master crea la rama master en el repositorio y sube los cambios
  10. Finalmente, cuando seguimos haciendo cambos ejecutamos: git push(luego de haber agregado los cambios y realizados commit)
